## Summary
A mux picks one input, a demux sends one input to one output.

## Lab Questions

### In plain English describe the function and use of a multiplexer.
A multiplexer picks one input and sends it out.

### In plain English describe the function and use of a demultiplexer.
A demultiplexer takes one input and sends it to one output.

### What other uses might these circuits have? (Think Shannon’s)
They help route and share data in digital systems.
